# Reference Overview

The Reference section documents **absolute, enforced rules** of the Apistry runtime.

Reference pages:
- describe behavior that is testable and non-negotiable
- avoid tutorials and walkthroughs
- act as lookup material once the system is understood

---

## Scope

Reference documentation covers:
- CLI command behavior
- Contract interpretation rules
- API semantics
- Query semantics
- Field and context rules

If you are learning Apistry, start with Overview, Getting Started, and Guides first.
