# SWAPI v1 <span style={{fontSize: '0.5em'}}>([Swagger](https://api.apistry.net/swagger-ui#/Swapi%20v1))</span>

This is a sample API for the original Star Wars API (SWAPI), which is a popular open API for Star Wars data. It
includes endpoints for retrieving information from the Star Wars universe.

Main differences from original SWAPI api are:

1. Base URL is `/v1/` instead of `/api/`
2. Pagination is offset based
3. All properties are queriable as filters (e.g. `?name=*R2*` to search for people with "R2" in their name)
